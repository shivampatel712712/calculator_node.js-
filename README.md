# Calculator Node.js

A simple calculator application built with Node.js that performs basic arithmetic operations.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Easy to use
- Lightweight Node.js project

## Installation

1. Clone the repository:

```bash
git clone https://github.com/shivampatel712712/calculator_node.js-.git
```

2. Navigate to the project folder:

```bash
cd calculator_node.js-
```

3. Install dependencies:

```bash
npm install
```

## Usage

Run the application:

```bash
node index.js
```

## Example

```text
Enter first number: 10
Enter second number: 5

Addition: 15
Subtraction: 5
Multiplication: 50
Division: 2
```
